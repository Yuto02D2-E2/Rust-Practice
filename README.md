## How to get start

Linux(WSL)ならこの一行で環境構築完了
```bash
# in terminal
$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

WinならC++ build toolsというものを入れた後，インストーラ(exe file)でインストールする必要がある(めんどくさい)
